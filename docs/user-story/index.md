# Historias de Usuario

Bienvenido a la sección de **Historias de Usuario** de Stable Experts. En este apartado documentamos los requisitos del sistema desde la perspectiva del usuario final, detallando el valor que aporta cada funcionalidad.

Las historias están organizadas por módulos funcionales para facilitar su lectura y seguimiento.

## Estructura de esta Sección

### [Gestión de Usuarios (Autenticación)](usuarios/US-001_Registro.md)
En esta categoría encontrará las historias relacionadas con el ciclo de vida de la cuenta de usuario y la seguridad del acceso. Incluye:

- **Registro**: Creación de cuentas nuevas.
- **Autenticación (Login)**: Acceso seguro al sistema.
- **Recuperación**: Restablecimiento de credenciales.
- **Cierre de Sesión**: Finalización segura de la actividad.
