# Casos de Uso

Bienvenido a la sección de Casos de Uso de **Stable Experts**. Aquí detallamos cómo las diferentes funcionalidades de la plataforma resuelven necesidades específicas para cada tipo de usuario.

El sistema está diseñado para integral la gestión del cuidado equino, conectando datos biométricos, operaciones diarias y decisiones estratégicas.

## Estructura de esta Sección

En esta documentación se describirán detalladamente los casos de uso para los diferentes procesos del sistema, abarcando desde la gestión operativa hasta la atención clínica.

- **[Actores del Sistema](Actores.md)**: Describe los perfiles de usuario que interactúan con la plataforma, desde veterinarios y entrenadores hasta propietarios y administradores. Entender estos roles es fundamental para comprender los flujos de trabajo descritos posteriormente.

- **[Gestión de Usuarios](CasosDeUso_Usuarios.md)**: Detalla los flujos relacionados con la administración de cuentas, autenticación, roles y permisos dentro de la plataforma.

## Objetivo
El objetivo principal es demostrar cómo **Stable Experts** transforma la gestión tradicional de establos en una operación digitalizada, predictiva y eficiente, priorizando siempre el bienestar animal.
